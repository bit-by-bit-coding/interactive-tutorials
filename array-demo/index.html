What is an array?

Arrays are special objects that store lists of data in order. In JavaScript, arrays come with functions that allow us to check their length, sort them, add new values to the end of them, etc.

ARRAY CONTENTS

The numbers in the blue boxes represent the array "indexes," which put the elements of the array in order. Arrays always start counting at 0, not 1. So the first element of the array is #0, the second is #1, and so on.
