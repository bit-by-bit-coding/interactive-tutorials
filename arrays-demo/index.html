Intro to Arrays

Arrays are ways to store many items in one variable.

For example, say you wanted to store the strings "dog", "cat", and "mouse", but you don't want to store them all in separate variables. You can use an array. Here is how you do it.

// This creates an array that is composed of the strings "dog", "cat", and "mouse", and assigns it to a variable called animals
let animals = ["dog", "cat", "mouse"];
// Notice the syntax of the array: it starts with an open bracket "[", and ends with a closing bracket "]". All the elements are separated by commas.

Adding Elements to Arrays

Now, this is great and all, but how is this any different from just assigning the strings to normal variables?

Well, say you woke up the next morning, and decided you want to store the word "hamster". If you were using normal variables, and assigning strings to variables as we normally do, then you would have to create a new variable each time you wanted to add a new animal. What if you went to the zoo, and wanted to store a hundred animals? What if you already defined the array, and you want the program to add more animals while it is running?

With arrays, you don't have to do that. All you need to do is use the push method, as shown:

let animals = ["dog", "cat", "mouse"];
// This adds "hamster" to the array, so the new array is ["dog", "cat", "mouse", "hamster"]
animals.push("hamster");
// Notice the syntax of the push method: you start with the name of the variable, in this case "animals", then write ".push".
// Then you enclose the element you want to add (in this case "hamster") in parantheses.

Now, whenever you want to store a new animal, you don't need to create a new variable! You can just add it to the same array!

Indexing Arrays

Arrays can do much more than store strings of animals, though. In fact, they can store any object - integers, booleans, even other arrays! This makes them extremely powerful.

However, once you have an array, how can you access the items inside of it? You do so using the index of the item. The index of an item in an array is the number assigned to it based on the order it appears in the array. For example, in our previous array, the index of "cat" is less than the index of "hamster" since "cat" is before "hamster" in the array.

The first element in an array starts with the index 0 (not 1!). The index increases by 1 for each element after. For example, in our previous array, "dog" has an index of 0, "cat" has an index of 1, "mouse" has an index of 2, and "hamster" has an index of 3.

Why does this matter so much? You can retrieve elements from an array using their index, as shown:

let animals = ["dog", "cat", "mouse"];
animals.push("hamster");
// Notice the syntax when you retrieve an element from an array. You start with the name of the array, then you open brackets "[", put the index of the element you want to retrieve, then close brackets "]".
console.log(animals[0]);
// This takes the element with index 0 (the first element) in the array animals (in this case "dog") and logs it to the console
console.log(animals[3]);
// This takes the element with index 3 (the fourth element) in the array animals (in this case "hamster") and logs it to the console
console.log(animals[1]);
// This takes the element with index 1 (the second element) in the array animals (in this case "cat") and logs it to the connsole

Now, you can store animals and retrieve them from an array!
