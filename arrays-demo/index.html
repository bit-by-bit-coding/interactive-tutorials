ARRAYS DEMO

What are Arrays?

Arrays are a way to store multiple ordered elements in one variable. They can be anything - strings, integers, floats, even other arrays!

let animals = ["dog", "cat", "mouse"]; // Defines an array called "animals" with "dog", "cat", and "mouse" in it, in that order
animals.push("hamster"); // Adds "hamster" to the end of the array
console.log(animals[0]); // Logs "dog" to the console
console.log(animals[3]); // Logs "hamster" to the console
animals.pop(); // Deletes the last element from the array ("hamster")
let deletedAnimal = animals.pop(); // Deletes the last element from the array ("mouse") and sets it to a variable "deletedAnimal"
