What is the display property?

The display property controls how content fits inside of a block-level element.

There are many display properties to choose from but the four most commons ones are:

  • block
  • inline
  • inline-block
  • none

The code snippet to the right shows how the content in the blue square below is being rendered. It is important to note, that by default, the overflow property, even when not specified in the CSS, is set to visible. Cycle through different overflow properties in given a time interval and change colors too.

.blue-square {

background-color: #1bb1de;

height: 320px;

width: 320px;

display: block;

}

more explanation

block: Displays an element as a block element (like p). It starts on a new line, and takes up the whole width

inline: Displays an element as an inline element (like span). Any height and width properties will have no effect

inline-block: Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values

none: The element is completely removed
