CSS DISPLAY DEMO

What is the display property?

The display property tells an HTML element how to render - or display - itself on the screen.

There are many display properties to choose from but the four most common ones are:

  • block
  • inline
  • inline-block
  • none

The code snippet to the right shows an example of how to set the display property of an element in CSS to some of the many display properties that are possible. It is important to note that each element has it's own default display value. A <div>, for example, defaults to a block display, while a <span> defaults to an inline display.

.blue-container {

background-color: blue;

display: block;

}

What does each display property do?

Below is a table explaining how four of the many display properties render elements on the screen.

PROPERTY NAME FUNCTION
BLOCK

Displays an element starting on a new line, taking up the whole width of the line by default. You can set the height and/or width values of the element.

Ex. <div>, <h1>, and <p>
INLINE

Displays an element on the same line as other elements, only taking up the width and height needed for the content inside to fit. You cannot set the width or height values of the element.

Ex. <input> and <span>
INLINE-BLOCK

Displays an element on the same line as other elements like an inline element, however you can set the height and/or width values of the element like you can with a block element.

Ex. a <div> with a specified height and/or width and a display value of inline-block
NONE

The element is not displayed and removed from the flow layout
