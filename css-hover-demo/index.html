CSS HOVER DEMO

What is the hover psuedo-class?

The CSS hover psuedo-class can be used to select and style an element while a user's mouse is hovered over that element.

To apply the hover psuedo-class to an element, simply append :hover to the element's selector.

The example below demonstrates how we can change the background color of an element using the hover psuedo-class.

How do you use hover?

In the first code snippet, we select the blue circle to the right using the class selector .blue-circle and style it to have a blue background by default.

.blue-circle {

background-color: blue;

height: 300px;

width: 300px;

border-radius: 50%;

}

In the second code snippet, we select the blue circle again, this time adding :hover to the end of the .blue-circle class selector and style it to have an orange background when a user hovers over it with their mouse.

.blue-circle:hover {

background-color: orange;

}
