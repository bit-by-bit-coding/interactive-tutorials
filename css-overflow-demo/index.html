What is the overflow property?

The overflow property controls how content fits inside of a block-level element.

The overflow property does this by using one of the four values below:

  • Visible
  • Scroll
  • Hidden
  • Auto

The code snippet to the right shows how the content in the blue square below is being rendered. It is important to note, that by default, the overflow property, even when not specified in the CSS, is set to visible.

Click on one of the overflow properties below to explore how the code snippet and content inside of the blue square changes with each property.

.blue-square {

background-color: #1bb1de;

height: 320px;

width: 320px;

overflow: visible;

}

Select an overflow property:

When visible is selected, the extra content "overflows" outside of the blue square.

When scroll is selected, the extra content is hidden inside of the blue square, and a scrollbar is added to the square in order to access the extra content.

When hidden is selected, the extra content is hidden by the blue square and inaccessible.

When auto is selected, if there is extra content, it is hidden inside of the blue square and a scrollbar is added to the square if necessary.
