What is the position property?

Did you know there is more than one way to position HTML elements on a page?

With the position property, you can specify how elements are positioned on a page relative to the page itself, or even to other elements! Below are the five values to utilize when positioning an element:

  • fixed
  • sticky
  • static
  • relative
  • absolute

The code snippet to the right shows an example of how to set the position property of an element in CSS to any of the five possible values. It is important to note that, by default, an element's position is static unless otherwise stated. Below, we'll explore more about each of the five position values.

div {

position: static;

}
