CSS POSITION DEMO

Information

Property Description
static Default value (follows the regular flow of the document)
relative Element is positioned relative to itself; does not affect other elemenets
absolute Element is positioned relative to its closest positioned* ancestor; affects other elements
fixed Element is positioned relative to the browser window and always stays in the same place
sticky Element is positioned either relative or fixed, depending on the user's scroll position

Instructions

Play around with the position properties of Bit by Bot and the balloon. Note that when changing the values of top/bottom and left/right, you are essentially adding pixels to that side, making the image move the other way.
