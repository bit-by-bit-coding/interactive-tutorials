CSS SELECTORS DEMO

    What is a CSS Selector?

  • CSS Selectors are used to target elements within an HTML document. A majority of the CSS selectors you will use, look just like HTML elements without the angle brackets.
  • You can use general selectors such as p, h1, body and div to target every single one of those elements on the page.
    For example:
    This is the "p" css selector. Creating this styling will target every single <p> element in the HTML document and change their text color to blue.
    This is also the correct syntax for CSS inside of a stylesheet: YOUR_SELECTOR { YOUR_STYLINGS; }. Every single HTML element can be targeted this way
    Note: Every style must be ended with a semi-colon like in the example when using an external stylesheet

    • Classes and ID's

  • General stylings are useful, but what if you wanted to target only one paragraph or div? This is where classes and ID's come into play
  • The two main rules that need to be remembered when using classes and ID's are:
    • Classes are identified using a " . ", ID's are identified using a " # "
    • Classes can be applied to multiple elements, ID's can only be applied to one element and must be uniquely named
  • In this example we are targeting two different paragraphs. One has a class of "blue-p" and the other has an ID of "red-p". This will change one paragraph element blue and the other red.
    Remember, ".blue-p" can be given to other elements to make them blue as well. "#red-p" cannot be assigned to another element and will only affect the element that has been given that ID.
    • Note: The syntax for applying styles to classes and ID's is exactly the same as applying stylings to general elements.
    Review: Classes are applied to elements using "class= CLASS_NAME", ID's are applied with "id= ID_NAME"

    Practice

  • In the editor below there is an internal style tag located at the top, to allow you to use CSS selectors to target elements on the page.
  • Practice targeting the paragraph elements on the page and change their color. Use the "h1" styling that is already present as a reference for the syntax.
  • Once you have the hang of selecting and styling general elements, try giving elements classes and ID's and try styling them by selecting their class name or ID name.
    • Note: Internal styling in an HTML document is generally considered not good practice, you should use an external stylesheet if you can
    Note: Remember to use a " ; " after each style. You cannot add mutliple styles without it.
