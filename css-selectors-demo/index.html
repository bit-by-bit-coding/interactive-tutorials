    What is a CSS Selector?

  • CSS Selectors are used to target elements within an HTML document. These look just like HTML elements without the angle brackets.

    • For example:
  • This is the correct syntax for CSS inside of a stylesheet: CSS_SELECTOR {YOUR_STYLINGS;}
  • This is the "p" general css selector. This will target every single <p> element and change their text color to blue.
    • Note: Every style must be ended with a semi-colon like in the example, when using an external stylesheet

    Classes and ID's

  • What if you wanted to target only one paragraph or a div? This is where classes and ID's come in.
  • Here is a table that shows the differences between Classes and ID's:
    • CLASSES
    Identified using a " . "
    Can be applied to multiple elements
    <p class="CLASS_NAME">
    ID's
    Identified using a " # "
    Can be applied to only one element
    <p id="ID_NAME">
  • In this example we are targeting three bit by bots. Two with a ".blue-bit-bot" class and the other with a "#gold-bit-bot" ID.
    Remember, ".blue-bit-bot" can be given to multiple bit-by-bots. "#gold-bit-bot" cannot.
    • Note: The syntax for applying styles to classes and ID's is identical to applying stylings with general elements
