    What is a CSS Selector?

  • CSS Selectors are used to target elements within an HTML document. A majority of the CSS selectors you will use look just like HTML elements without the angle brackets.
  • You can use general selectors such as p, h1, body and div to target every single one of those elements on the page.

    This is the "p" css selector. Creating this styling will target every single <p> element in the HTML document and change their text color to blue.
    This is also the correct syntax for CSS inside of a stylesheet: YOUR_SELECTOR { YOUR_STYLINGS }. Every single HTML element can be targeted this way

    • Classes and ID's

  • General stylings are useful, but what if you wanted to target only one paragraph or div? This is where classes and ID's come into play
  • The two main rules that need to be remembered when using classes and ID's:
    • Classes are identified using a " . ", ID's are identified using a " # "
    • Classes can be applied to multiple elements, ID's can only be applied to one element and must be unique
  • Take note that the syntax for applying styles to classes and ID's is exactly the same as applying stylings to general elements.
  • In this example we are targeting two different paragraphs. One has a class of "first-p" and the other has an ID of "second-p". This will change one paragraph element blue and the other red.
    Remember, ".blue-p" can be given to other elements to make them blue as well. "#red-p" cannot be assigned to another element and will only affect the element that has been given that ID.
