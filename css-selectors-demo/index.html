    What is a CSS Selector?

  • CSS Selectors are used to target elements within an HTML document. These look just like HTML elements without the angle brackets.

  • This is the correct syntax for CSS inside of a stylesheet: YOUR_SELECTOR { YOUR_STYLINGS; }.

    • Note: Every style must be ended with a semi-colon like in the example, when using an external stylesheet

    Classes and ID's

  • What if you wanted to target only one paragraph or div? This is where classes and ID's come in.
  • The two main rules with classes and ID's are:
    • Classes are identified using a " . ", ID's are identified using a " # "
    • Classes can be applied to multiple elements, ID's can only be applied to one element
  • In this example we are targeting three robots. Two with a "blue-bot" class and the other with a "red-bot" ID.
    Remember, ".blue-bot" can be given to multiple robots. "#red-bot" cannot.
    • Note: The syntax for applying styles to classes and ID's is exactly the same as applying stylings to general elements.
    Review: Classes are applied to elements using "class= CLASS_NAME", ID's are applied with "id= ID_NAME"
