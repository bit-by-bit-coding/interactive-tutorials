Flex-Box Containers

In order to make a flex-box container, simply set the "display" CSS property of a div tag to "flex". Here is a summary of the CSS properties associated with a flex-box container:

Property Description Values
flex-direction Defines which direction the container stacks the items column, column-reverse, row, row-reverse
flex-wrap Defines whether the flex items should wrap wrap, nowrap, wrap-reverse
justify-content Aligns the items along the primary axis (aligns horizontally if flex-direction is row and aligns vertically if flex-direction is column) center, flex-start, flex-end, space-around, space-between
align-items Aligns items along the secondary axis (aligns vertically if flex-direction is row and aligns horizontally if flex-direction is column) center, flex-start, flex-end, stretch, baseline
align-content Used to align the flex lines (the rows/columns of flex items) space-between, space-around, stretch, center, flex-start, flex-end

Flex-Box Items

Any div tags inside the flex container become flex items. You can adjust the individual flex properties of each flex item using these properties:

Property Description Values
order Specifies the order the flex items are shown in Any integer
flex-grow Specifies how fast a flex item will grow relative to the other flex items Any positive number
flex-shrink Specifies how fast a flex item will shrink relative to the other flex items Any positive number
flex-basis Specifies the initial length of a flex item Any positive integer number of pixels
align-self Specifies the alignment for a specific flex item. Overrides the default align-items property in the container center, flex-start, flex-end, stretch, baseline
