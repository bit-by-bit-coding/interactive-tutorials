For Loops

Here is a breakdown of for loop syntax:

for
(
let i = *numberOne*;
i < *numberTwo*;
i = i + 1
){
*repeated code*
}

The first part of the for loop is the word for. This tells the computer that we are creating a for loop.
The next part of the for loop is the condition (the thing inside the parentheses). This condition is made up of three parts:

  1. Declaring a condition variable:
    let i = *numberOne*;
  2. Declaring the condition:
    i < *numberTwo*;
  3. Declaring how our condition variable changes each time the for loop runs:
    i = i + 1

We can tell how many time the code is repeated by subtracting

*numberOne*
by
*numberTwo*

Here are some examples of functions:

for
(
let i = 0;
i < 3;
i = i + 1
){
alert("Hello World!");
}

for
(
let i = 5;
i < 10;
i = i + 1
){
alert(i);
}

for
(
let i = 3;
i >= 0;
i = i - 1
){
if(i===0){
alert("Blast off!");
} else {
alert(i);
}
}
