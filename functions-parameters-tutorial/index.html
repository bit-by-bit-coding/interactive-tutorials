Functions

Functions are a fundemental part of JavaScript. They allow us to store lines of code to use later. Unlike variables, functions can store multiple lines of code including: alerts, variables, prompts, and much more!

There are three different parts of a function: the function name, declaring the function, and the contained code.

code
}

Parameters

Parameters are another more complex part of functions. They are basically variables that are made specifically for the function. These parameters are declared inside parentheses. The value of these parameters are then assigned when calling/running the function once again, inside the parentheses.

You can make as many parameters as you want, they just need to be inside the parentheses, and separted by a comma.

Parameter syntax:

let functionName
=
function(
parameter1, parameter2, parameter3
)
{
code
}

Here are some examples of functions with parameters:

let mathEquation

x
