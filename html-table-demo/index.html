HTML TABLE DEMO

What is an HTML table?

The <table> element in HTML allows data to be represented by a two-dimensional table of rows and columns. The intersection of a table's rows and columns form table cells, and these cells contain the data displayed in the table.

To define a table cell, place the data between a <td> and a </td> tag.

Once you have defined a table cell, place this cell between a table row's <tr> and </tr> tags. Note, you can have multiple rows in a table, just make sure that the number of cells in each row are the same!

Finally, place your table row or rows between a <table> and a </table> tag to form a complete table.

As a bonus, if you'd like your table to have headers, create and place another table row before all other table rows in your table. In this newly created table row, place a <th> and a </th> tag with the title of the heading inbetween. Note, make sure that the number of <th> tags is the same as the number of <td> in each table row! This will ensure that each column has a heading.

Table Examples

Example 1

The HTML to the right creates a table with one row, hence the single <tr>, a two columns. Each <td> inside the <tr> represents a separate column.

HTML

<table>

<tr>

<td>Row 1 Column 1</td>

<td>Row 1 Column 2</td>

</tr>

</table>

Outputted Table

Row 1 Column 1 Row 1 Column 2

Example 2

If you wanted to add another row to the table in Example 1, then simply add another <tr> with a same amout of <td> tags as the first <tr> inside of the <table>.

If you wanted to add another column to the table, then add another <td> in each <tr>.

HTML

<table>

<tr>

<td>Row 1 Column 1</td>

<td>Row 1 Column 2</td>

<td>Row 1 Column 3</td>

</tr>

<tr>

<td>Row 2 Column 1</td>

<td>Row 2 Column 2</td>

<td>Row 2 Column 3</td>

</tr>

</table>

Outputted Table

Row 1 Column 1 Row 1 Column 2 Row 1 Column 3
Row 2 Column 1 Row 2 Column 2 Row 2 Column 3

Example 3

If you wanted to add table headings to Example 2, then you would simply create another table row before all other table rows in your table. In this newly created table row, place a <th> and a </th> tag with the title of the heading inbetween, and make sure that the number of <th> tags matches the number of <td> in each table row.

Hooray! The final outputted table is shown to the right!

HTML

<table>

<tr>

<th>Heading 1</th>

<th>Heading 2</th>

<th>Heading 3</th>

</tr>

<tr>

<td>Row 1 Column 1</td>

<td>Row 1 Column 2</td>

<td>Row 1 Column 3</td>

</tr>

<tr>

<td>Row 2 Column 1</td>

<td>Row 2 Column 2</td>

<td>Row 2 Column 3</td>

</tr>

</table>

Outputted Table

Heading 1 Heading 2 Heading 3
Row 1 Column 1 Row 1 Column 2 Row 1 Column 3
Row 2 Column 1 Row 2 Column 2 Row 2 Column 3
