IF / ELSE DEMO

What is an If / Else Statement?

It allows you to create a condition based on some information. If that information is TRUE, it will run something. If that information is FALSE it will run something else.

Example

if (true) {
alert("if condition is true, this alert will run");
} else {
alert("if condition is anything else but true, this alert will run ");
}

What if we need more than 2 conditions?

Sometimes you want your code to have more than a true and false condition. So then what? If you need more than 2 conditions, you can add an else if statement.

Example

if (favoriteColor === "blue") {
alert("Your favorite color is blue.");
} else if (favoriteColor === "purple") {
alert("Your favorite color is purple.");
} else {
alert("Your favorite color is something other than blue or purple.");
}
