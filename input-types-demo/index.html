Input Types Demos

Forms

  • Forms contains information passes from the user to a server.
  • We use the tag <form> to define the form.

Input Types

  • We use the tag <input> to define several input data alternatives.
  • <input> appears inside of the <form> tag.
  • The general format is <input type="ALTERNATIVES">, where ALTERNATIVES can be text, radio, checkbox, button, number, color, date or others.

Input Type Text

<input type="text"> defines a one-line text input fields.

Text Input Example

Input Type Radio

<input type="radio"> is used to let the user select exactly one option from a pre-defined set of options.

Radio Input Example

Input Type Checkbox

<input type="checkbox"> allows the user to select or unselect one or more option from a pre-defined set of options.

Checkbox Input Example

Input Type Button

<input type="button"> defines a simple push button, which can be programmed to control custom functionality on any event such as, click event.

Button Input Example

Input Type Number

<input type="number"> creates a numeric input. You can also set restrictions on the maximum and minimum number accepted.

Number Input Example

Input Type Color

<input type="color"> creates a color input fields. Depending on browser support, a color picker can appear in the input field.

Color Input Example

Input Type Date

<input type="text"> creates a date input fields. Depending on browser support, a date picker can appear in the input field.

Date Input Example
