Input Types Demos

Forms

  • Forms contains information passes from the user to a server.
  • We use the tag <form> to define the form.

Input Types

  • We use the tag <input> to define several input data alternatives.
  • <input> appears inside of the <form> tag.

General Format

/* The HTML element <form> creates a form to collect user input. */

/* You can also add an attribute onsubmit to execute a JavaScript when the form is submitted. */

<form>


/* The HTML element <label> is used to define labels for input element. */

/* The for attribute should be equal to the id attribute in <input> element to bind them together. */

<label for="name">Enter your name:</label>


/* The HTML element <input> defines input types. */

/* Notice that the id attribute in <input> element is equal to the for attribute in <label> element. */

/* Each input field must have a name attribute to be submitted. */

<input type="text" id="name" name="name">


/* The HTML element <input type="submit"> defines a submit button. */

/* The value attribute specifies the default value of <input> element. */

<input type="submit" value="Submit!">


</form>
