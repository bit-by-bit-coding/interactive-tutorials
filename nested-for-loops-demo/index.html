NESTED FOR LOOPS DEMO

Introduction

A for loop executes the code block inside its body during each cycle of the loop.

We may let the code block be whatever we want depending on the task at hand; sometimes the code block may contain another for loop, this is called nested for loops.

For a pair of nested for loops, the inner for loop completely runs itself during each cycle of the outer loop.

Example

Imagine if we want to know the favorite movie of each bit by bot below. One of the most straight forward ways to accomplish this goal is to start from the first column, ask the bots sitting in each row, then move on to the next column, until we have recorded the favorite movies of every bot.

Example Code

Let's try to capture this method in code. Since, on the big picture, we ask the bots column by column, the outer for loop should correspond to the number of columns, which is 4.

Then for each column, we ask each bot row by row, so the inner for loop corresponds to the number of rows, which is 3. The code block of the inner loop is where we ask the bot sitting at that position.

for (let i = 0; i < 4; i++) {
  for (let j = 0; j < 3; j++) {

    // Here is the code block that does the
    action of asking the student, who sits at
    this position, what is their favorite movie

  }
}
