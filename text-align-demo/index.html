What is Text-Align?

The CSS text-align property allows you to align text. It can take on four different values:


Value Description
Left Aligns text to the left
Right Aligns text to the right
Center Centers the text
Justify Stretches the lines so that each line has equal width

Interactive

Try manipulating the values on the left and see how it affects the result on the right!
