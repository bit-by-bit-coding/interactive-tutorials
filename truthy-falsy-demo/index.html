TRUTHY & FALSY VALUES DEMO

In JavaScript, boolean (true/false) types and comparisons are not the only forms of truthy and falsy values. But first, what are truthy and falsy values?

As the names suggest...

  • Truthy expressions result in the boolean value true
  • Falsy expressions result in the boolean value false
Here are the falsy values:

Falsy Value Description
false If the boolean variable is set to be false, then it will evaluate to false.
0 If the value of the number is 0, then it will be considered as falsy.
"" or '' (Empty String) If the string is blank, then it will evaluate to false.
null If there is no value at all, then JavaScript will return a value called null, which will also be considered as falsy.
undefined If a variable is declared but not assigned a value, then it will return a value called undefined and will also evaluate to false.
NaN If there is a problem with the number (e.g. divide a string by a number), then NaN (Not a Number) will be returned and will evaluate to false.

Values other than the ones listed above are truthy values.
